# Steps to run the frontend app:

  * run `npm install`
  * open 3 different terminals and execute only ONE command from the below in each terminal:

    * `npm run start-m1`
    * `npm run start-m2`
    * `npm run start-s1`

  * open 3 different browsers, each browser with one of the below urls:
    * `http://localhost:4200`
    * `http://localhost:4201`
    * `http://localhost:4202`
